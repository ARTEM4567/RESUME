Шлюз для Woocommerce на основе Яндекс Деньги

== Description ==

Позволяет организовать в интернет магазине на базе Woo приём оплат на личный кошелёк Woo

Если возникнут проблемы, [сообщите об ошибке](http://www.coderun.ru/)


== Installation ==
1. Убедитесь что у вас установлена посленяя версия плагина [WooCommerce](http://www.woothemes.com/woocommerce)
2. Распакуйте архив и загрузите "yandex-noip-woocommerce" в папку ваш-домен/wp-content/plugins
3. Активируйте плагин
4. Перейдите в пункт меню "WooCommerce" - "Настройка" - "Платежи" для настройки дополнения
